Selamat Datang di Aplikasi Manajemen Sertifikat Eletronik Pemerintah Provinsi Jawa Tengah.

Berikut ini beberapa poin penting yang harus diperhatikan sebagai Kebijakan Privasi penggunaan aplikasi ini agar dapat berjalan sesuai dengan fungsinya:

1. Aplikasi ini membutuhkan akses koneksi internet agar dapat terhubung dengan server kami.

2. Aplikasi ini membutuhkan akses penyimpanan perangkat hanya pada saat anda memilih file untuk kelengkapan berkas pendukung Sertifikat Eletronik.

3. Aplikasi ini juga terhubung dengan e-Personal System dari BKD Provinsi Jawa Tengah untuk mendapatkan akses data berbasis NIP anda. Bertujuan sebagai kelengkapan administrasi untuk mendapatkan informasi Sertifikat Eletronik anda.

4. Selain dari akses yang disebutkan diatas, aplikasi ini tidak membutuhkan akses lainnya yang tidak diperlukan.
